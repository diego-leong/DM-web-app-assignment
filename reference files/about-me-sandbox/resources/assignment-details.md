Build a clean, professional “About Me” page using Bootstrap 5.3. This is open-ended, so explore Bootstrap — but make sure you meet the requirements below.

Your page must follow semantic HTML standards (use proper tags like `<header>`, `<main>`, `<section>`, `<footer>`), primarily use Bootstrap for layout and styling, and only use custom CSS when needed (in an external stylesheet). Your page will be run through HTML and CSS code validators and a contrast checker, so make sure your colors are accessible and readable.

## Required Content
Include:
- [ ] A photo of you with descriptive alt text
- [ ] 1–2 paragraphs about you
- [ ] A list with at least 2 skills, projects, or work experiences
- [ ] Links to your Web App and IoT final project (use `#` as a placeholder if needed)
- [ ] A LinkedIn/GitHub/Social link that opens in a new tab (or `#` if you don’t have one yet)
- [ ] Include Open Graph meta tags in the `<head>`

## Required Styling
- [ ] Show that you can use Bootstrap and CSS by including:
- [ ] Different font color, family, and size (including one imported Google font)
- [ ] A background color somewhere on the page
- [ ] Resized image styling
- [ ] Bootstrap’s grid system
- [ ] At least two Bootstrap components (like a navbar, buttons, cards, etc.)
- [ ] At least one Bootstrap utility class

When you’re done, paste your page URL in the comments, submit it in Classroom, and add the link to the HTML/CSS/JavaScript Worksheet.

Think of this as your first real “developer calling card.” If someone Googled you, this could be the page they see. Make it clean, make it thoughtful, and make it something you’d be proud to show a future employer.
